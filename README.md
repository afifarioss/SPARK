SPARK is the first Farcaster-native intelligence agent that discovers alpha using FID reputation scoring (EigenTrust/PoQ), channel-specific analysis, and Frame execution. It turns the Farcaster firehose into actionable intelligence.

---

## ✨ Features

### 🧠 FID Reputation Layer
- EigenTrust/PoQ scoring algorithm
- Trust levels: Legendary → Expert → Builder → Community Member → Newcomer
- Not based on follower count - based on *quality* of engagement

### 📡 Channel-Specific Intelligence
- Deep analysis of `/base`, `/degen`, `/clanker`, `/crypto`, `/nft`
- Channel-specific keyword matching
- Custom alpha scoring per channel

### 🖼️ Frame Execution
- "See it → Trade it" loop inside the feed
- Auto-detects Frame actions: mint, trade, vote, interact
- Generates actionable Frame URLs

### ⚡ x402 Ready
- Pay-per-use micro-payments
- Free tier: 10 sparks/day
- Seamless USDC payments

### 🔓 Open Source
- Transparent scoring algorithm
- Community auditable
- Built on Farcaster's PoQ FIP
